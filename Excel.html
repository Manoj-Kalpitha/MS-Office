Microsoft Excel

Microsoft Excel is one of the most popular spreadsheet programs available in the Microsoft Office Suite. It allows us to record data in the form of tables. Also, it is a beneficial program when it comes to data analyzing, adding graphs, inserting pivot tables, performing calculations, macro programming, and more. For example, we can create an Excel spreadsheet to calculate weekly expenses, generate salary slips, accounting, and interactively sort/ filter data by different criteria.
